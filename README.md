# 项目简介 java499

本项目是基于 Python 的农家乐酒店管理系统，旨在提供便捷的信息管理和交流平台。

## 技术栈

- 后端：Python、Django 框架
- 前端：jQuery
- 数据库：MySQL 5.7.33

## 功能模块

### 前台管理

- 农家乐信息
- 美食信息
- 住宿信息
- 活动信息
- 论坛交流
- 系统公告
- 个人中心

### 后台管理

- 农家乐管理
- 美食信息管理
- 住宿信息管理
- 活动信息管理
- 客房预订管理
- 用户评价管理
- 论坛交流
- 管理员管理
- 系统管理

## 系统角色

- 管理员
- 用户

## 运行环境

- Python 环境推荐：Python 3.6 或以上版本
- 数据库环境：MySQL 5.7.33

## 部署步骤

1. 搭建 Python 运行环境。
2. 安装 Django 框架。
3. 配置 MySQL 数据库。
4. 导入数据库结构。
5. 运行系统。
